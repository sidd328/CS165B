Homework assignment 2, problem 5

There are 2 train datasets given to you (training1.txt, training2.txt), along with 2 corresponding test datasets for you to check your program (testing1.txt, testing2.txt).

Data files are text files with the following format:

D N1 N2 N3

f1_1_1_value f1_1_2_value ... f1_1_D_value

f1_2_1_value f1_2_2_value ... f1_2_D_value

...

f1_N1_1_value f1_N1_2_value ... f1_N1_D_value

f2_1_1_value f2_1_2_value ... f2_1_D_value

f2_2_1_value f2_2_2_value ... f2_2_D_value

...

f2_N2_1_value f2_N2_2_value ... f2_N2_D_value

f3_1_1_value f3_1_2_value ... f3_1_D_value

f3_2_1_value f3_2_2_value ... f3_2_D_value

...

f3_N3_1_value f3_N3_2_value ... f3_N3_D_value

In the first line, D is the number of features in each instance, N1, N2, N3 are the number of examples in class 1, 2, and 3, respectively.

Starting from the second line, each line represents a data point with D features. The first N1 lines are data points from class 1, the next N2 lines are data points from class 2, and the last N3 lines are data points from class 3.

So there are N1+N2+N3+1 lines in a data file in total.

Each feature component is a real number - e.g., f1_3_2_value is the second feature value of the third data point from class 1.

While the data we give you is resticted to D=3, your program should be able to handle any number of features D and with any number of training and testing instances N1, N2, N3.

A new dataset will be used along with the 2 given datasets to check your program. You can assume that there are always 3 classes.
